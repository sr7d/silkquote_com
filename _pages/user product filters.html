Enable a user to sell a specific product by creating a "User Product Permission" record via dataloder, automation, or the related list. permalink: /user-product-filters/ image: image_caption: ---

After using the install buttons, please take the following steps:

  1. Navigate to: Setup --> Installed Packages --> Salesforce CPQ --> Configure --> Plugins.
  2. Confirm you are not currently using the "Product Search Plugin" (very few are).
  3. Paste "cpqdev.ProductSearch" to your Product Search Plugin field.
  4. Save.
