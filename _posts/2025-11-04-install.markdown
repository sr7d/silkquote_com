## Installation Instructions

Great! Now that you have installed the managed package, follow the steps below to set up your org.
If you have an issues feel free to reach out to Support!

### Step 1: Navigate to the SilkQuote App

Open the SilkQuote App from your Salesforce App Launcher.

![SilkQuote App](/images/install1-min.png)

### Step 2: Assign the Permission Set

Grant users access to SilkQuote by assigning the permission set.

![Permissions](/images/install2-min.png)

![Permissions](/images/install3-min.png)

### Step 3: Upload and Configure Your Company Logo

Upload your company logo to Salesforce Static Resources and configure it in the Quote Template.

1. Click the **+** icon to add your logo (from the DEFAULT template, or whichever template you want to use)

   ![Static Resources](/images/install4-min.png)

2. Click **New** and upload your logo image file

   ![Upload Logo](/images/install5-min.png)

3. Open the newly created Static Resource record and click **View File**

   ![View File](/images/install6-min.png)

4. Copy the resource name from the URL (the text after `/resource/`)

   ![Copy Resource Name](/images/install7-min.png)

5. Paste this resource name into the **Logo** field in the Quote Template settings and click **Save**

   ![Configure Logo](/images/install8-min.png)

### Step 4: Add the Generate Quote Button

Add the Generate Quote button to your Opportunity page layout so users can access the quote generation wizard.

1. Navigate to the Opportunity page Lightning page layout of your choice

   ![Page Layout](/images/install9-min.png)

2. Add the **Generate Quote Document** button to the page

   ![Add Button](/images/install10-min.png)

3. Add the **Email Quote Document** button to the page

   ![Add Button](/images/install10-min.png)

4. Add the **SilkQuote Live Panel** to the page

   ![Add SilkQuote Live Panel](/images/install10-min.png)

5. Save your changes

### Step 5: Create a Site

Go to Setup and create a site.

1. Name and activate the site

   ![Create Site](/images/siteCreate.png)

2. Copy the site URL for later use

### Step 6: Configure the Site

Open the site you created and configure it.

1. Click the site name and select edit

2. Update the active site home page to be: Quote Review Page

   ![Configure Site](/images/siteHomePage.png)

3. Save

### Step 7: Edit Public Access Settings

Open the site's public access settings.

   ![Public Access](/images/guestProfileClassAndPageAccess.png)

1. Enable silkquote_QuoteReviewController

2. Enable silkquote_QuoteReviewPage

3. Edit Opportunity Access

![Public Access](/images/guestProfileOppFieldAccess.png)

4. Grant Edit Access to:
- Quote Document URL
- Quote Response At
- Quote Response Token
- Quote Status


### Step 8: Update Quote Templates

For each template you want to support customer acceptance or change requests:

1. Paste the site URL you copied earlier into the **Quote Review Site URL** field and save

   ![Page Layout](/images/updateTemplateSite.png)

### You're All Set!

Congratulations! When users click the Generate Quote button, they will be presented with the Quote Generation Wizard.
